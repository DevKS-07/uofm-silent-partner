# Silent Partner

Our solution is a React Native mobile app for real-time social coaching. It's basically a networking wingman right in your pocket. Instead of a chaotic crowd, you get a highly curated list of valuable connections. We pinpoint exactly who you need to meet out of thousands of attendees, and give you the real-time coaching and exact scripts to start the conversation.

## Tech Stack

**Frontend:** React Native(Expo), TypeScript, Expo Router for file-based navigation, React Native Reanimated for animations, and Zod for schema validation.

**Backend:** Firebase Firestore (database), Firebase Auth (authentication), and LinkedIn OAuth for social login.

**API:** Expo Audio + expo-audio-studio for audio capture and playback, WebRTC for real-time peer connections.

**AI:** OpenAI Realtime API for live conversational AI.
